# R79.1 style board

## 主題與構圖

「中央沉默、兩翼燃燼」：直式 `start.png` 是唯一焦點；atmosphere 只延伸桌機兩側世界，不建立第二個 hero。中央 32% 保持煤黑、低細節、低對比，左右才出現遠景公路、廢墟輪廓與少量餘火。

## 自我檢核

- 不採 generic neon sci-fi、爆炸電影海報或滿版高細節廢墟。
- 唯一視覺風險是把中央做成近乎「缺席」的暗帶；這正用來保護現行 portrait key art，與 1366×700 版面契約一致。
