| 宣稱 | 證據 | 判定 |

**行為語意（正確性）**

1. **預留：** 低優先（0）永遠留出 25% 空位給後續中／高優先 — 清雜兵前若曳光／塵已狂噴，不會先把池塞死。  
2. **回收：** 僅 priority 2（smoke／flash）在 **free=0** 時可踢 priority 0／1；debris／fire／shard 仍為 1，**不會**互搶。  
3. **Boss ×3 壓力：** 理論需求仍可 &gt;96，但煙／閃可擠掉曳光與環境粒 — V1／V2 的「煙先被砍」主債 **實質緩解**，非完全消失。

| 殘留 | 級別 |
|------|------|
| fire 紋理非 priority 2（Boss 火層仍可能輸給後到的 smoke） | **P3** — 通常可接受（煙為殘留語意） |
| 滿池 eviction O(n) 線性掃 | **P3** — n≤96 可忽略 |
| 線性 alpha、無 hold 峰值 | **P3** — 跨版本債，本輪未承諾 |

> **結論：** V2 ③ 池側 **PASS**；是 R66 起最重要的爆炸「不廉價化」工程修復。

## 4. 剩餘最大缺口（收官只標一個主因）

### 主缺口：**景深「有結構、無體量」＋ 非 land 仍扁平**

| 維度 | 現況 | 為何是最大 |
|------|------|------------|
| land 剪影 | 遠 7–13px、中 5–9px 方塊 | 手機縮圖上幾乎變成路肩噪點，**定裝世界感回收率低** |
| sea／air | 無 `drawDepthLayers` 同級 | 四環境宣傳拼圖只 land 有厚度，產品印象碎裂 |
| 對照 | space 已有 starDust；land 有路＋薄剪影 | 中間兩環境成視覺窪地 |

**為何不是別的債當「最大」**

| 候選 | 為何退居次位 |
|------|----------------|
| Raster 無視 damage／wreck | 低血煙已補敘事主幹；本體換圖成本高、CP 中 |
| 爆炸 alpha／Boss×3 | 優先級＋scorch 已把「廉價」壓到 P3 |
| 受擊 rim／敵彈 kind | 可讀已過門檻，屬打磨 |
| Kenney scorch 紋理未接 | 橢圓已有錨點，邊際 |

**若只准再動一刀（建議，非本輪施工）**

1. **放大 land 遠／中剪影高度**到接近 V2 規格（遠 ~22px、中 ~40px），維持現有視差與分級 — 零新資產、改常數與 path。  
2. **同一 API 給 sea／air 最小參數表**（V2 §3.5）— 產品觀感立刻「四關都是世界」。  
3. 車體 damage 狀態列為 **可選第二刀**（煙已夠用時可後置）。
